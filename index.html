Mview-master

Reverse Engineer MView 3D File Format（逆向工程.Mview 3d文件格式）

软件下载

在GitHub上搜索Mview ， 找到Python写的majimboo/mviewer插件。、

  • Mviewer GitHub URL(网址) : https://github.com/majimboo/mviewer

模型提取（网页）

在Artstation封面，若有左上角小方块图标的，就代表该页面有模型媒体文件。

像这种中间有个圆圈，还有一个小三角的是GZ文件，我们一般是（提取）不了的。

但是像 MarmosetToolbag3 link的Media文件是可以提取出来的（.Mview）

步骤：
刷新页面，直到所有网页都加载完毕（除了八猴媒体文件先不点开）
按F12调出网页源代码，点击Network

等所有页面加载完后，我们可以点击Clear，清除网页加载好的Log

之后我们再点击加载Toolbag媒体文件，就能清晰地找到它加载的Log

之后我们从Log中找到.mview的文件

右键，并保存至本地
完成

模型提取（本地）

我们把GitHub下载好的Mview-Master解压，打开Noesis.exe。

模型导入


按步骤将模型导入（或直接拖拽进Noesis）

  • 蓝色区域为过滤器
  • 绿色区域为模型信息
  • 黄色区域为显示模式

模型导出

  • 选择需导出的媒体文件，并点击Export即可
  • 若没有勾选导出文件
  • 它会提示你在列表视图中未选择可导出文件。是否要从正在预览的文件中导出
  • 选择是即可

软件简评

优点：

  • 很明显，Mview 3d文件逆向工程这块做的还是很好的
  • 内置的Data Viewer也能便捷的参看模型信息

    缺点：
  • Mview-master缺少Gloss通道，无法导出Glossiness光泽度贴图。
